# 17 - 反射

## 1. 反射是什么

反射可以在程序运行时查看变量的**类型**和**值**。

常用包：

```go
import "reflect"
```

常用函数：

- `reflect.TypeOf`：获取类型信息。
- `reflect.ValueOf`：获取值信息。

## 2. 获取类型

`TypeOf` 可以获取变量的具体类型。

```go
package main

import (
	"fmt"
	"reflect"
)

func main() {
	name := "Go"

	t := reflect.TypeOf(name)

	fmt.Println(t)
	fmt.Println(t.Kind())
}
```

输出：

```text
string
string
```

**重点：**`Type()` 更具体，`Kind()` 更粗略。

## 3. Type 和 Kind 的区别

自定义类型的 `Type` 和 `Kind` 可能不一样。

```go
package main

import (
	"fmt"
	"reflect"
)

type MyString string

func main() {
	var name MyString = "Go"

	t := reflect.TypeOf(name)

	fmt.Println(t)
	fmt.Println(t.Kind())
}
```

输出：

```text
main.MyString
string
```

**重点：**`Type` 是真实类型名，`Kind` 是底层分类。

## 4. 判断类型分类

可以用 `Kind()` 判断是切片、map、结构体还是字符串。

```go
package main

import (
	"fmt"
	"reflect"
)

func printKind(value any) {
	t := reflect.TypeOf(value)

	switch t.Kind() {
	case reflect.Slice:
		fmt.Println("切片")
	case reflect.Map:
		fmt.Println("map")
	case reflect.Struct:
		fmt.Println("结构体")
	case reflect.String:
		fmt.Println("字符串")
	}
}

func main() {
	printKind("Go")
	printKind([]int{1, 2})
	printKind(map[string]int{"age": 18})
}
```

## 5. 获取值

`ValueOf` 可以获取变量的值。

```go
package main

import (
	"fmt"
	"reflect"
)

func main() {
	value := reflect.ValueOf("Go")

	fmt.Println(value)
	fmt.Println(value.String())
	fmt.Println(value.Type())
}
```

**注意：**不同类型要用不同方法取值，例如 `String()`、`Int()`、`Bool()`。

## 6. 按类型读取值

读取前先判断 `Kind()` 更安全。

```go
package main

import (
	"fmt"
	"reflect"
)

func printValue(value any) {
	v := reflect.ValueOf(value)

	switch v.Kind() {
	case reflect.Int:
		fmt.Println(v.Int())
	case reflect.String:
		fmt.Println(v.String())
	default:
		fmt.Println(v.Interface())
	}
}

func main() {
	printValue(100)
	printValue("Go")
}
```

**重点：**`Interface()` 可以把反射值转回普通 `any`。

## 7. 读取结构体字段

反射可以遍历结构体字段。

```go
package main

import (
	"fmt"
	"reflect"
)

type Student struct {
	Name string
	Age  int
}

func main() {
	s := Student{Name: "枫枫", Age: 18}

	t := reflect.TypeOf(s)
	v := reflect.ValueOf(s)

	for i := 0; i < t.NumField(); i++ {
		field := t.Field(i)
		value := v.Field(i)

		fmt.Println(field.Name, field.Type, value)
	}
}
```

## 8. 读取结构体 tag

结构体 tag 常用于 JSON、表单、数据库映射。

```go
package main

import (
	"fmt"
	"reflect"
)

type Student struct {
	Name string `json:"name"`
	Age  int    `json:"age"`
}

func main() {
	t := reflect.TypeOf(Student{})

	for i := 0; i < t.NumField(); i++ {
		field := t.Field(i)
		fmt.Println(field.Name, field.Tag.Get("json"))
	}
}
```

**重点：**`field.Tag.Get("json")` 可以读取 `json` 标签。

## 9. 修改结构体字段

要通过反射修改值，必须传指针，再调用 `Elem()`。

```go
package main

import (
	"fmt"
	"reflect"
)

type Student struct {
	Name string
}

func main() {
	s := Student{Name: "go"}

	v := reflect.ValueOf(&s).Elem()
	field := v.FieldByName("Name")

	if field.CanSet() {
		field.SetString("Go")
	}

	fmt.Println(s.Name)
}
```

**重点：**能不能修改，先看 `CanSet()`。

## 10. 根据 tag 修改字段

可以根据 tag 决定要不要修改字段。

```go
package main

import (
	"fmt"
	"reflect"
	"strings"
)

type Student struct {
	Name string `upper:"true"`
	City string
}

func main() {
	s := Student{Name: "go", City: "beijing"}

	t := reflect.TypeOf(s)
	v := reflect.ValueOf(&s).Elem()

	for i := 0; i < t.NumField(); i++ {
		field := t.Field(i)
		value := v.Field(i)

		if field.Tag.Get("upper") == "true" && value.CanSet() {
			value.SetString(strings.ToUpper(value.String()))
		}
	}

	fmt.Println(s)
}
```

## 11. 调用方法

反射也可以按方法名调用方法。

```go
package main

import (
	"fmt"
	"reflect"
)

type Student struct {
	Name string
}

func (s Student) Say(msg string) {
	fmt.Println(s.Name, msg)
}

func main() {
	s := Student{Name: "Go"}

	method := reflect.ValueOf(s).MethodByName("Say")
	method.Call([]reflect.Value{
		reflect.ValueOf("hello"),
	})
}
```

**注意：**传参时必须包装成 `reflect.Value`。

## 12. 什么时候用反射

适合使用反射：

- 写框架。
- 写通用工具。
- 处理结构体 tag。
- 做序列化、参数绑定、ORM 等底层能力。

不适合使用反射：

- 普通业务逻辑。
- 明确知道类型的地方。
- 为了少写几行代码强行使用。

## 13. 易错点

- **`TypeOf` 看类型，`ValueOf` 看值**。
- **`Type` 是具体类型，`Kind` 是底层分类**。
- **修改值必须传指针并调用 `Elem()`**。
- **修改字段前先判断 `CanSet()`**。
- **调用方法时参数要转成 `reflect.Value`**。
- **反射可读性较差，性能也比普通代码低**。
