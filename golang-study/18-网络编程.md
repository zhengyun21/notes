# 18 - 网络编程

## 1. 常见网络通信方式

Go 常见网络编程场景：

- **TCP**：面向连接，可靠传输。
- **HTTP**：浏览器和接口最常用。
- **RPC**：像调用本地函数一样调用远程服务。
- **WebSocket**：客户端和服务端保持长连接，适合实时消息。

## 2. TCP 是什么

TCP 是传输层协议，特点是**面向连接**、**可靠**、**字节流**。

常见面试点：

- 三次握手：建立连接。
- 四次挥手：关闭连接。

## 3. TCP 服务端

服务端监听端口，等待客户端连接。

```go
package main

import (
	"fmt"
	"io"
	"net"
)

func handleConn(conn net.Conn) {
	defer conn.Close()

	buf := make([]byte, 1024)

	for {
		n, err := conn.Read(buf)
		if err == io.EOF {
			fmt.Println("客户端断开")
			return
		}
		if err != nil {
			fmt.Println(err)
			return
		}

		fmt.Println("收到：", string(buf[:n]))
		conn.Write([]byte("ok"))
	}
}

func main() {
	listener, err := net.Listen("tcp", "127.0.0.1:8080")
	if err != nil {
		fmt.Println(err)
		return
	}
	defer listener.Close()

	fmt.Println("TCP server: 127.0.0.1:8080")

	for {
		conn, err := listener.Accept()
		if err != nil {
			fmt.Println(err)
			continue
		}

		go handleConn(conn)
	}
}
```

**重点：**每个连接通常开一个 goroutine 处理。

## 4. TCP 客户端

客户端连接服务端，发送消息，再读取响应。

```go
package main

import (
	"fmt"
	"net"
)

func main() {
	conn, err := net.Dial("tcp", "127.0.0.1:8080")
	if err != nil {
		fmt.Println(err)
		return
	}
	defer conn.Close()

	conn.Write([]byte("hello"))

	buf := make([]byte, 1024)
	n, _ := conn.Read(buf)

	fmt.Println(string(buf[:n]))
}
```

## 5. HTTP 服务端

HTTP 服务端用 `net/http`。

```go
package main

import (
	"fmt"
	"net/http"
)

func hello(w http.ResponseWriter, r *http.Request) {
	fmt.Fprintln(w, "hello Go")
}

func main() {
	http.HandleFunc("/hello", hello)

	fmt.Println("HTTP server: http://localhost:8080/hello")
	http.ListenAndServe(":8080", nil)
}
```

访问：

```text
http://localhost:8080/hello
```

## 6. 读取 HTTP 请求

可以通过 `req.Method` 判断请求方法。

```go
package main

import (
	"fmt"
	"io"
	"net/http"
)

func user(w http.ResponseWriter, r *http.Request) {
	if r.Method == http.MethodPost {
		body, _ := io.ReadAll(r.Body)
		fmt.Println(string(body))
		fmt.Fprintln(w, "post ok")
		return
	}

	fmt.Fprintln(w, "get ok")
}

func main() {
	http.HandleFunc("/user", user)
	http.ListenAndServe(":8080", nil)
}
```

**重点：**`r.Body` 是请求体，读完后通常不能再读第二次。

## 7. HTTP 客户端

用 `http.Get` 发送简单 GET 请求。

```go
package main

import (
	"fmt"
	"io"
	"net/http"
)

func main() {
	resp, err := http.Get("http://localhost:8080/hello")
	if err != nil {
		fmt.Println(err)
		return
	}
	defer resp.Body.Close()

	data, _ := io.ReadAll(resp.Body)
	fmt.Println(string(data))
}
```

**注意：**读取完响应后，要关闭 `resp.Body`。

## 8. RPC 是什么

RPC 可以理解为：客户端像调用函数一样调用服务端方法。

Go 标准库提供了 `net/rpc`。

## 9. RPC 服务端

```go
package main

import (
	"fmt"
	"net"
	"net/http"
	"net/rpc"
)

type Args struct {
	A int
	B int
}

type Server struct{}

func (Server) Add(args Args, reply *int) error {
	*reply = args.A + args.B
	return nil
}

func main() {
	rpc.Register(Server{})
	rpc.HandleHTTP()

	listener, err := net.Listen("tcp", ":8080")
	if err != nil {
		fmt.Println(err)
		return
	}

	http.Serve(listener, nil)
}
```

**重点：**RPC 方法必须是导出方法，参数和返回值也要符合 `net/rpc` 规则。

## 10. RPC 客户端

```go
package main

import (
	"fmt"
	"net/rpc"
)

type Args struct {
	A int
	B int
}

func main() {
	client, err := rpc.DialHTTP("tcp", "127.0.0.1:8080")
	if err != nil {
		fmt.Println(err)
		return
	}

	var reply int
	err = client.Call("Server.Add", Args{A: 1, B: 2}, &reply)
	if err != nil {
		fmt.Println(err)
		return
	}

	fmt.Println(reply)
}
```

## 11. WebSocket 是什么

WebSocket 是长连接协议，适合聊天室、实时通知、实时数据推送。

Go 标准库没有直接提供 WebSocket 服务，常用第三方库：

```powershell
go get github.com/gorilla/websocket
```

## 12. WebSocket 服务端

最小 echo 服务端：收到什么，就回什么。

```go
package main

import (
	"fmt"
	"net/http"

	"github.com/gorilla/websocket"
)

var upgrader = websocket.Upgrader{}

func echo(w http.ResponseWriter, r *http.Request) {
	conn, err := upgrader.Upgrade(w, r, nil)
	if err != nil {
		fmt.Println(err)
		return
	}
	defer conn.Close()

	for {
		msgType, msg, err := conn.ReadMessage()
		if err != nil {
			return
		}

		conn.WriteMessage(msgType, msg)
	}
}

func main() {
	http.HandleFunc("/ws", echo)
	http.ListenAndServe(":8080", nil)
}
```

## 13. WebSocket 客户端

```go
package main

import (
	"fmt"

	"github.com/gorilla/websocket"
)

func main() {
	conn, _, err := websocket.DefaultDialer.Dial("ws://127.0.0.1:8080/ws", nil)
	if err != nil {
		fmt.Println(err)
		return
	}
	defer conn.Close()

	conn.WriteMessage(websocket.TextMessage, []byte("hello"))

	_, msg, _ := conn.ReadMessage()
	fmt.Println(string(msg))
}
```

**注意：**WebSocket 需要先启动服务端，再运行客户端。

## 14. 易错点

- **TCP 是字节流，不会自动按消息分包**。
- **TCP 服务端处理多个连接时，通常用 goroutine**。
- **HTTP 响应体读完后要关闭 `Body`**。
- **`r.Body` 读完后不能随便重复读取**。
- **RPC 方法和参数类型需要符合 `net/rpc` 要求**。
- **WebSocket 不是普通 HTTP 请求，而是 HTTP 升级后的长连接**。
- **简单练习尽量用短例子，先跑通流程再扩展功能**。
