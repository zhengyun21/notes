# 14 - 文件操作

## 1. 一次性读取文件

小文件可以用 `os.ReadFile` 一次性读取。

```go
package main

import (
	"fmt"
	"os"
)

func main() {
	data, err := os.ReadFile("hello.txt")
	if err != nil {
		fmt.Println(err)
		return
	}

	fmt.Println(string(data))
}
```

**重点：**读取到的是 `[]byte`，转成字符串用 `string(data)`。

## 2. 获取当前 Go 文件路径

`runtime.Caller` 可以获取调用位置。

```go
package main

import (
	"fmt"
	"runtime"
)

func currentFile() string {
	_, file, _, _ := runtime.Caller(0)
	return file
}

func main() {
	fmt.Println(currentFile())
}
```

参数含义：

- `0`：当前函数。
- `1`：调用当前函数的地方。
- `2`：再上一层调用者。

## 3. 分块读取

大文件可以分块读，避免一次性读入内存。

```go
package main

import (
	"fmt"
	"io"
	"os"
)

func main() {
	file, err := os.Open("hello.txt")
	if err != nil {
		fmt.Println(err)
		return
	}
	defer file.Close()

	buf := make([]byte, 4)

	for {
		n, err := file.Read(buf)
		if err == io.EOF {
			break
		}
		if err != nil {
			fmt.Println(err)
			return
		}

		fmt.Print(string(buf[:n]))
	}
}
```

**重点：**`Read` 返回实际读取的字节数 `n`，要用 `buf[:n]`。

## 4. 按行读取

按行读推荐用 `bufio.Scanner`。

```go
package main

import (
	"bufio"
	"fmt"
	"os"
)

func main() {
	file, err := os.Open("hello.txt")
	if err != nil {
		fmt.Println(err)
		return
	}
	defer file.Close()

	scanner := bufio.NewScanner(file)

	for scanner.Scan() {
		fmt.Println(scanner.Text())
	}

	if err := scanner.Err(); err != nil {
		fmt.Println(err)
	}
}
```

## 5. 按单词读取

`Scanner` 可以指定分割方式。

```go
scanner := bufio.NewScanner(file)
scanner.Split(bufio.ScanWords)

for scanner.Scan() {
	fmt.Println(scanner.Text())
}
```

常见分割方式：

- `bufio.ScanLines`：按行。
- `bufio.ScanWords`：按单词。
- `bufio.ScanRunes`：按字符。
- `bufio.ScanBytes`：按字节。

## 6. 一次性写入文件

小内容可以用 `os.WriteFile` 写入。

```go
package main

import (
	"fmt"
	"os"
)

func main() {
	err := os.WriteFile("file.txt", []byte("hello Go"), 0644)
	if err != nil {
		fmt.Println(err)
	}
}
```

**注意：**文件不存在会创建，文件存在会覆盖原内容。

## 7. 追加写入文件

追加内容可以用 `os.OpenFile`。

```go
package main

import (
	"fmt"
	"os"
)

func main() {
	file, err := os.OpenFile("file.txt", os.O_CREATE|os.O_APPEND|os.O_WRONLY, 0644)
	if err != nil {
		fmt.Println(err)
		return
	}
	defer file.Close()

	_, err = file.WriteString("追加一行\n")
	if err != nil {
		fmt.Println(err)
	}
}
```

常用打开方式：

- `os.O_RDONLY`：只读。
- `os.O_WRONLY`：只写。
- `os.O_RDWR`：读写。
- `os.O_CREATE`：不存在就创建。
- `os.O_APPEND`：追加写。
- `os.O_TRUNC`：打开时清空。

## 8. 文件权限

权限主要在 Linux / macOS 生效，Windows 下不明显。

常见权限：

- `0644`：所有者可读写，其他人只读。
- `0666`：所有人可读写。
- `0755`：所有者可读写执行，其他人可读可执行。
- `0777`：所有人可读写执行。

数字含义：

```text
读 r = 4
写 w = 2
执行 x = 1
```

例如 `7 = 4 + 2 + 1`，表示读、写、执行都有。

## 9. 文件复制

复制文件可以用 `io.Copy`。

```go
package main

import (
	"fmt"
	"io"
	"os"
)

func main() {
	src, err := os.Open("file.txt")
	if err != nil {
		fmt.Println(err)
		return
	}
	defer src.Close()

	dst, err := os.Create("copy.txt")
	if err != nil {
		fmt.Println(err)
		return
	}
	defer dst.Close()

	n, err := io.Copy(dst, src)
	fmt.Println(n, err)
}
```

**重点：**`io.Copy(dst, src)` 是从 `src` 复制到 `dst`。

## 10. 读取目录

`os.ReadDir` 可以读取目录下的文件和文件夹。

```go
package main

import (
	"fmt"
	"os"
)

func main() {
	entries, err := os.ReadDir(".")
	if err != nil {
		fmt.Println(err)
		return
	}

	for _, entry := range entries {
		info, _ := entry.Info()
		fmt.Println(entry.Name(), info.Size())
	}
}
```

**注意：**`Size()` 的单位是字节。

## 11. 易错点

- **读文件返回的是 `[]byte`**。
- **打开文件后记得 `defer file.Close()`**。
- **分块读取要使用 `buf[:n]`**。
- **`os.WriteFile` 会覆盖原文件内容**。
- **追加写要用 `os.O_APPEND`**。
- **文件权限一般写 `0644`、`0755` 这种八进制格式**。
- **`io.Copy(dst, src)` 的目标在前，来源在后**。
