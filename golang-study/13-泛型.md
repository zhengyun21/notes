# 13 - 泛型

## 1. 泛型是什么

泛型就是把**类型也当作参数**传给函数或类型。

适合场景：逻辑一样，只是数据类型不同。

## 2. 泛型函数

普通函数只能固定一种类型。

```go
func addInt(a, b int) int {
	return a + b
}
```

泛型函数可以支持多种类型。

```go
package main

import "fmt"

func add[T int | float64](a, b T) T {
	return a + b
}

func main() {
	fmt.Println(add(1, 2))
	fmt.Println(add(1.2, 2.3))
}
```

**重点：**`T` 是类型参数，`int | float64` 是类型约束。

## 3. any

`any` 表示任意类型，等价于 `interface{}`。

```go
package main

import "fmt"

func printValue[T any](value T) {
	fmt.Println(value)
}

func main() {
	printValue("Go")
	printValue(100)
}
```

**注意：**`any` 只能表示“接收任意类型”，不能直接做 `+`、`-` 等运算。

## 4. 泛型结构体

泛型结构体适合做通用返回结构。

```go
package main

import "fmt"

type Response[T any] struct {
	Code int
	Msg  string
	Data T
}

type User struct {
	Name string
}

func main() {
	res := Response[User]{
		Code: 0,
		Msg:  "成功",
		Data: User{Name: "枫枫"},
	}

	fmt.Println(res.Data.Name)
}
```

**重点：**`Response[User]` 表示这个响应里的 `Data` 是 `User` 类型。

## 5. 泛型切片

可以定义自己的泛型切片类型。

```go
package main

import "fmt"

type MySlice[T any] []T

func main() {
	names := MySlice[string]{"枫枫", "知道"}
	numbers := MySlice[int]{1, 2, 3}

	fmt.Println(names)
	fmt.Println(numbers)
}
```

## 6. 泛型 map

map 的 key 必须是可比较类型，所以泛型 key 也要有限制。

```go
package main

import "fmt"

type MyMap[K comparable, V any] map[K]V

func main() {
	user := MyMap[string, string]{
		"name": "枫枫",
	}

	fmt.Println(user["name"])
}
```

**重点：**`comparable` 表示这个类型可以用 `==`、`!=` 比较，适合当 map 的 key。

## 7. 什么时候用泛型

适合用泛型：

- 逻辑重复，只是类型不同。
- 写通用容器，比如列表、结果、响应结构。
- 写通用工具函数。

不适合用泛型：

- 只有一个具体类型。
- 用了泛型反而更难读。
- 为了炫技硬套泛型。

## 8. 易错点

- **泛型从 Go 1.18 开始支持**。
- **`T` 只是类型参数名，可以换成别的名字**。
- **`any` 等价于 `interface{}`**。
- **需要运算时，不能只写 `any`，要写具体类型约束**。
- **map 的泛型 key 通常写 `comparable`**。
