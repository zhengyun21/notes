# 01 - 输入与输出

## 学习目标

- 掌握 Go 中最基础的**输出**方式：`fmt.Println` 和 `fmt.Printf`。
- 知道输出前如何准备数据：**变量**、**常量**、**零值**。
- 理解 `fmt.Scan` 的基础用法，以及为什么输入时要写 `&变量名`。

## 1. 先完成一次最简单的输出

Go 的输出通常来自 `fmt` 包。最常用的是 **`fmt.Println`**，它会在输出末尾自动换行。

```go
package main

import "fmt"

func main() {
	fmt.Println("你好，Go")
}
```

**注意：**Go 区分大小写，应该写 `fmt.Println`，不是 `fmt.println`。

## 2. 输出前先准备变量

变量可以先声明再赋值，也可以在声明时直接赋值。

```go
package main

import "fmt"

func main() {
	var name string
	name = "zzy"

	var city string = "Beijing"
	var language = "Go"

	fmt.Println(name)
	fmt.Println(city)
	fmt.Println(language)
}
```

在函数内部，还可以使用**短变量声明** `:=`。

```go
package main

import "fmt"

func main() {
	name := "zzy"
	fmt.Println(name)
}
```

**重点：**`:=` 只能在函数内部使用，不能在函数外部声明全局变量。

## 3. 理解变量的零值

如果变量已经声明但没有手动赋值，Go 会给它一个对应类型的**零值**。

常见零值：

- `int` 的零值是 `0`
- `float64` 的零值是 `0`
- `string` 的零值是空字符串 `""`
- `bool` 的零值是 `false`

```go
package main

import "fmt"

func main() {
	var age int
	var name string
	var passed bool

	fmt.Println(age)
	fmt.Println(name)
	fmt.Println(passed)
}
```

**重点：**零值不是“没有值”，而是 Go 自动给出的默认值。

## 4. 区分局部变量和全局变量

定义在函数内部的变量叫**局部变量**。局部变量声明后必须使用，否则编译会报错。

定义在函数外部的变量叫**全局变量**。全局变量可以暂时不使用。

```go
package main

import "fmt"

var userName = "枫枫知道"

func main() {
	name := "枫枫"
	fmt.Println(name)
}
```

**注意：**上面的 `userName` 是全局变量，即使没有在 `main` 中使用，也不会报错。

## 5. 一次定义多个变量

多个变量类型相同时，可以写在一起。

```go
package main

import "fmt"

func main() {
	var name1, name2, name3 string
	name1 = "枫枫"
	name2 = "知道"
	name3 = "Go"

	fmt.Println(name1, name2, name3)
}
```

也可以在声明时直接赋值。

```go
package main

import "fmt"

func main() {
	var firstName, lastName = "枫枫", "知道"
	a, b := 1, 2

	fmt.Println(firstName, lastName)
	fmt.Println(a, b)
}
```

**注意：**多变量赋值时，左边变量数量和右边值的数量要一致。

## 6. 使用常量保存不会变的数据

常量使用 `const` 定义，定义时必须赋值，之后不能再修改。

```go
package main

import "fmt"

func main() {
	const siteName string = "枫枫知道"

	fmt.Println(siteName)
}
```

下面这种写法会报错，因为常量不能再次赋值。

```go
const siteName = "枫枫知道"
siteName = "知道"
```

**重点：**变量适合保存会变化的数据，常量适合保存固定不变的数据。

## 7. 了解基础命名规范

Go 中的命名既影响可读性，也影响包外访问权限。

- **首字母大写**：可以被包外访问，例如 `UserName`
- **首字母小写**：只能在当前包内访问，例如 `userName`

```go
var UserName = "可以被包外访问"
var userName = "只能在当前包内访问"
```

**注意：**初学时先优先写清楚名字，例如 `userName`、`age`、`totalCount`。

## 8. 使用 Printf 做格式化输出

`fmt.Printf` 可以使用占位符控制输出格式。

```go
package main

import "fmt"

func main() {
	fmt.Printf("%v\n", "你好")
	fmt.Printf("%v %T\n", "枫枫", "枫枫")
	fmt.Printf("%d\n", 3)
	fmt.Printf("%.2f\n", 1.25)
	fmt.Printf("%s\n", "哈哈哈")
	fmt.Printf("%#v\n", "")
}
```

常用占位符：

| 占位符 | 含义 |
| --- | --- |
| `%v` | 按默认格式输出任意值 |
| `%T` | 输出值的类型 |
| `%d` | 输出整数 |
| `%.2f` | 输出小数，并保留 2 位小数 |
| `%s` | 输出字符串 |
| `%#v` | 按 Go 语法格式输出，适合观察空字符串等细节 |

**注意：**`fmt.Printf` 不会自动换行，需要自己加 `\n`。

## 9. 使用 Sprintf 得到格式化后的字符串

`fmt.Sprintf` 不会直接输出内容，而是把格式化后的结果作为字符串返回。

```go
package main

import "fmt"

func main() {
	message := fmt.Sprintf("你好，%s", "Go")
	fmt.Println(message)
}
```

**重点：**需要“拼好字符串再使用”时，优先考虑 `fmt.Sprintf`。

## 10. 从终端读取输入

`fmt.Scan` 可以从终端读取用户输入，并把结果保存到变量中。

```go
package main

import "fmt"

func main() {
	var name string

	fmt.Println("请输入您的名字：")
	fmt.Scan(&name)

	fmt.Println("你输入的名字是：", name)
}
```

**注意：**`fmt.Scan(&name)` 中的 `&` 表示把变量地址传进去。后面学习指针时会进一步理解它。

## 11. 本节易错点

- **函数名大小写不能写错**：`fmt.Println` 正确，`fmt.println` 错误。
- **局部变量声明后必须使用**，否则程序无法通过编译。
- **`:=` 只能在函数内部使用**。
- **`fmt.Printf` 不自动换行**，需要写 `\n`。
- **`fmt.Scan` 读取变量时要加 `&`**。
- **常量定义后不能修改**。
