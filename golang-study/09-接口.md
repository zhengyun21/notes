# 09 - 接口

## 1. 接口是什么

接口是一组**方法签名**的集合，只定义方法，不写具体实现。

```go
type Animal interface {
	Sing()
	Jump()
	Rap()
}
```

方法签名包含：

- 方法名
- 参数
- 返回值

## 2. 实现接口

一个类型只要实现了接口中的**全部方法**，就自动实现了这个接口。

```go
package main

import "fmt"

type Animal interface {
	Sing()
	Jump()
	Rap()
}

type Chicken struct {
	Name string
}

func (c Chicken) Sing() {
	fmt.Println(c.Name, "唱")
}

func (c Chicken) Jump() {
	fmt.Println(c.Name, "跳")
}

func (c Chicken) Rap() {
	fmt.Println(c.Name, "rap")
}

func main() {
	var animal Animal
	animal = Chicken{Name: "ik"}

	animal.Sing()
	animal.Jump()
	animal.Rap()
}
```

**重点：** Go 不需要写 `implements`。方法对上了，就实现了接口。

## 3. 接口值

接口变量里保存的是：

```text
具体类型 + 具体值
```

例如：

```go
var animal Animal
animal = Chicken{Name: "ik"}
```

此时 `animal` 中保存的是：

```text
具体类型：Chicken
具体值：Chicken{Name: "ik"}
```

**注意：**接口本身不能绑定方法。方法是绑定到具体类型上的。

## 4. 类型断言

使用接口时，如果想拿到具体类型，可以用类型断言。

```go
package main

import "fmt"

type Animal interface {
	Sing()
}

type Chicken struct{}

func (Chicken) Sing() {
	fmt.Println("chicken 唱")
}

func main() {
	var animal Animal = Chicken{}

	chicken, ok := animal.(Chicken)
	fmt.Println(chicken, ok)
}
```

**重点：** 推荐使用两个返回值：`value, ok := obj.(Type)`。

## 5. 单返回值断言

单返回值断言如果类型不匹配，会直接 panic。

```go
// 不推荐：类型不对会 panic
chicken := animal.(Chicken)
fmt.Println(chicken)
```

更安全的写法：

```go
chicken, ok := animal.(Chicken)
if ok {
	fmt.Println(chicken)
}
```

## 6. type switch

如果要判断多种具体类型，可以用 `type switch`。

```go
package main

import "fmt"

type Animal interface {
	Sing()
}

type Chicken struct{}
type Cat struct{}

func (Chicken) Sing() {
	fmt.Println("chicken 唱")
}

func (Cat) Sing() {
	fmt.Println("cat 唱")
}

func PrintAnimal(obj Animal) {
	switch obj.(type) {
	case Chicken:
		fmt.Println("鸡")
	case Cat:
		fmt.Println("猫")
	default:
		fmt.Println("未知动物")
	}

	obj.Sing()
}

func main() {
	PrintAnimal(Chicken{})
	PrintAnimal(Cat{})
}
```

## 7. 空接口

空接口没有任何方法，所以任意类型都实现了空接口。

```go
var value interface{}

value = 1
value = "hello"
value = true
```

Go 1.18 之后，`any` 是 `interface{}` 的别名。

```go
var value any

value = "Go"
fmt.Println(value)
```

**重点：**空接口能接收任何类型，但使用具体值时通常需要类型断言。

## 8. 易错点

- **接口只定义方法，不写实现**。
- **类型实现接口不需要显式声明**。
- **必须实现接口的全部方法，才算实现接口**。
- **接口变量保存具体类型和具体值**。
- **单返回值类型断言失败会 panic**。
- **空接口 `interface{}` 可以接收任意类型**。
- **`any` 等价于 `interface{}`**。
