# 05 - 循环

## 1. for 基本循环

Go 只有一种循环关键字：**`for`**。

```go
package main

import "fmt"

func main() {
	for i := 0; i < 5; i++ {
		fmt.Println(i)
	}
}
```

结构：

```go
for 初始化; 条件; 操作 {
	// 循环体
}
```

## 2. 死循环

省略条件就是死循环。

```go
package main

import (
	"fmt"
	"time"
)

func main() {
	for {
		time.Sleep(1 * time.Second)
		fmt.Println(time.Now().Format("2006-01-02 15:04:05"))
	}
}
```

**重点：**Go 的时间格式模板固定写法是 `2006-01-02 15:04:05`。

## 3. 用 for 实现 while

Go 没有 `while`，可以用只有条件的 `for` 实现。

```go
package main

import "fmt"

func main() {
	i := 0
	sum := 0

	for i <= 100 {
		sum += i
		i++
	}

	fmt.Println(sum)
}
```

## 4. 用 for 实现 do while

Go 没有 `do while`，可以用死循环加 `break` 实现。

```go
package main

import "fmt"

func main() {
	i := 0
	sum := 0

	for {
		sum += i
		i++

		if i == 101 {
			break
		}
	}

	fmt.Println(sum)
}
```

**重点：** 这类写法会先执行一次循环体，再判断是否退出，因为是先 do 再 while

## 5. 遍历切片

切片有顺序，可以用下标遍历。

```go
package main

import "fmt"

func main() {
	names := []string{"zzy", "yzz"}

	for i := 0; i < len(names); i++ {
		fmt.Println(i, names[i])
	}
}
```

更常用的是 `range`。

```go
package main

import "fmt"

func main() {
	names := []string{"枫枫", "知道"}

	for index, name := range names {
		fmt.Println(index, name)
	}
}
```

**重点：**遍历切片时，`range` 返回的是 `index` 和 `value`。

## 6. 遍历 map

遍历 map 时，`range` 返回的是 `key` 和 `value`。

```go
package main

import "fmt"

func main() {
	scores := map[string]int{
		"age":   24,
		"price": 1000,
	}

	for key, value := range scores {
		fmt.Println(key, value)
	}
}
```

**注意：**map 是无序的，遍历顺序不固定。

## 7. break 和 continue

`break` 用于跳出当前循环。

```go
package main

import "fmt"

func main() {
	for i := 0; i < 10; i++ {
		if i == 5 {
			break
		}

		fmt.Println(i)
	}
}
```

`continue` 用于跳过本轮循环，直接进入下一轮。

```go
package main

import "fmt"

func main() {
	for i := 0; i < 5; i++ {
		if i == 2 {
			continue
		}

		fmt.Println(i)
	}
}
```

## 8. 易错点

- **Go 没有 `while` 和 `do while`，都用 `for` 实现**。
- **死循环要有明确退出条件，否则程序会一直运行**。
- **遍历切片时，`range` 返回 `index, value`**。
- **遍历 map 时，`range` 返回 `key, value`**。
- **map 遍历顺序不固定**。
- **`break` 是退出循环，`continue` 是跳过本轮**。
