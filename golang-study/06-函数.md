# 06 - 函数

## 1. 函数是什么

函数是一段**可复用代码块**，用于完成特定任务。

函数通常包含：

- **参数**：传入函数的数据。
- **返回值**：函数执行后返回的结果。

## 2. 定义和调用函数

使用 `func` 定义函数。

```go
package main

import "fmt"

func sayHello() {
	fmt.Println("hello")
}

func main() {
	sayHello()
}
```

**重点：**调用函数时写 `函数名()`。

## 3. 函数参数

函数可以接收参数。

```go
package main

import "fmt"

func add(n1 int, n2 int) {
	fmt.Println(n1 + n2)
}

func main() {
	add(1, 2)
}
```

参数类型相同，可以合并写。

```go
func add(n1, n2 int) {
	fmt.Println(n1 + n2)
}
```

## 4. 可变参数

可变参数用 `...` 表示，适合接收不固定数量的参数。

```go
package main

import "fmt"

func add(nums ...int) {
	sum := 0

	for _, num := range nums {
		sum += num
	}

	fmt.Println(sum)
}

func main() {
	add(1, 2, 3)
	add(10, 20, 30, 40)
}
```

**重点：**`nums ...int` 在函数内部可以当作 `[]int` 使用。

## 5. 返回值

函数可以没有返回值，也可以有一个或多个返回值。

```go
package main

import (
	"errors"
	"fmt"
)

func fun1() {
	return
}

func fun2() int {
	return 1
}

func fun3() (int, error) {
	return 0, errors.New("错误")
}

func main() {
	fun1()
	fmt.Println(fun2())
	fmt.Println(fun3())
}
```

**注意**：多个返回值常用于同时返回结果和错误，例如 `(int, error)`。

## 6. 命名返回值

返回值可以提前命名。

```go
package main

import "fmt"

func getName() (name string) {
	name = "abc"
	return
}

func main() {
	fmt.Println(getName())
}
```

`return` 没写具体值时，会返回命名返回值当前的值。

**注意**：裸 `return` 虽然能用，但代码长了会降低可读性，初学时少用。

## 7. 匿名函数

匿名函数就是没有名字的函数，可以赋值给变量。

```go
package main

import "fmt"

func main() {
	add := func(a, b int) int {
		return a + b
	}

	fmt.Println(add(1, 2))
}
```

## 8. 高阶函数

函数也可以作为值保存到变量、切片或 map 中。

```go
package main

import "fmt"

func main() {
	login := func() {
		fmt.Println("登录")
	}

	quit := func() {
		fmt.Println("退出")
	}

	register := func() {
		fmt.Println("注册")
	}

	funcMap := map[int]func(){
		1: login,
		2: quit,
		3: register,
	}

	num := 1
	funcMap[num]()
}
```

**重点：**`map[int]func()` 表示 key 是 `int`，value 是一个没有参数、没有返回值的函数。

## 9. 闭包

闭包是**函数 + 它记住的外部变量**。

```go
package main

import "fmt"

func counter() func() int {
	count := 0

	return func() int {
		count++
		return count
	}
}

func main() {
	next := counter()

	fmt.Println(next())
	fmt.Println(next())
	fmt.Println(next())
}
```

`counter` 执行完后，内部的 `count` 仍然会被返回的匿名函数记住。

## 10. 闭包示例：延迟求和

函数可以返回另一个函数。

```go
package main

import (
	"fmt"
	"time"
)

func awaitAdd(t int) func(...int) int {
	time.Sleep(time.Duration(t) * time.Second)

	return func(nums ...int) int {
		sum := 0

		for _, num := range nums {
			sum += num
		}

		return sum
	}
}

func main() {
	add := awaitAdd(2)
	fmt.Println(add(1, 2, 3))
}
```

执行过程：

- `awaitAdd(2)` 先等待 2 秒。
- 返回一个匿名求和函数。
- `add(1, 2, 3)` 调用返回的函数，结果是 `6`。

## 11. 值传递

Go 函数参数默认是**值传递**。函数里改的是副本，不影响外面的变量。

```go
package main

import "fmt"

func change(num int) {
	num = 2
}

func main() {
	num := 20

	change(num)

	fmt.Println(num)
}
```

输出仍然是 `20`。

## 12. 指针参数

如果要在函数里修改外部变量，可以传变量地址。

```go
package main

import "fmt"

func change(num *int) {
	*num = 2
}

func main() {
	num := 20

	change(&num)

	fmt.Println(num)
}
```

**重点：**

- `&`：取变量地址。
- `*`：根据地址取值，也叫解引用。

## 13. 易错点

- **函数定义用 `func`**。
- **参数类型相同可以合并写**：`func add(a, b int)`。
- **可变参数写在参数列表最后**。
- **多个返回值常用于返回结果和错误**。
- **命名返回值配合裸 `return` 可以用，但不要滥用**。
- **Go 默认是值传递**。
- **想修改外部变量，要传指针**。
