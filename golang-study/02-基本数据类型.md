# 02 - 基本数据类型

## 学习目标

- 认识 Go 常见的**基本数据类型**。
- 理解整数、浮点数、布尔值、字符串、字符相关类型的基础用法。
- 掌握每种类型的**零值**和常见易错点。

## 1. 基本数据类型总览

Go 中常见的基本数据类型包括：

| 类型分类 | 常见类型 |
| --- | --- |
| 整数型 | `int`、`int8`、`int16`、`int32`、`int64`、`uint`、`uint8` 等 |
| 浮点型 | `float32`、`float64` |
| 复数型 | `complex64`、`complex128` |
| 布尔型 | `bool` |
| 字符串 | `string` |
| 字符相关 | `byte`、`rune` |

**重点：**Go 是强类型语言，变量一旦确定类型，就不能直接当作其他类型使用。

## 2. 整数型

整数型用于保存没有小数部分的数字。

```go
package main

import "fmt"

func main() {
	var n1 uint8 = 2
	var n2 uint16 = 2
	var n3 uint32 = 2
	var n4 uint64 = 2
	var n5 uint = 2

	var n6 int8 = 2
	var n7 int16 = 2
	var n8 int32 = 2
	var n9 int64 = 2
	var n10 int = 2

	fmt.Println(n1, n2, n3, n4, n5)
	fmt.Println(n6, n7, n8, n9, n10)
}
```

整数类型可以先记住这几条：

- **带 `u` 表示无符号整数**，只能保存 `0` 和正整数。
- 类型名后面的数字表示**二进制位数**，例如 `int8` 是 8 位。
- `uint8` 的别名是 **`byte`**，一个字节等于 8 个 bit。
- `int` 和 `uint` 的大小和平台有关，通常在 64 位系统上是 64 位。

### 整数范围怎么理解

以 `uint8` 为例，8 个二进制位都用来存数据，所以范围是：

```text
0 ~ 2^8 - 1
也就是 0 ~ 255
```

以 `int8` 为例，最高位要用来表示正负，所以实际范围是：

```text
-2^7 ~ 2^7 - 1
也就是 -128 ~ 127
```

**注意：**最大值要减 `1`，因为数字从 `0` 开始计数。就像一分钟有 60 秒，但秒数最大显示到 `59`。

## 3. 数字的默认类型

如果没有显式指定整数类型，Go 通常会把整数推断为 **`int`**。

```go
package main

import "fmt"

func main() {
	n := 100
	fmt.Printf("%v 的类型是 %T\n", n, n)
}
```

可以使用 `math` 包查看常见数值范围。

```go
package main

import (
	"fmt"
	"math"
)

func main() {
	fmt.Println(math.MaxInt)
	fmt.Println(math.MaxFloat32)
	fmt.Println(math.MaxFloat64)
}
```

**注意：**超过类型范围的值会导致溢出或编译错误，学习时不要硬塞超过上限的数字。

## 4. 浮点型

浮点型用于保存小数。Go 支持两种浮点类型：

- `float32`：范围约为 `3.4e38`
- `float64`：范围约为 `1.8e308`

```go
package main

import "fmt"

func main() {
	var price float32 = 12.5
	var score float64 = 98.75

	fmt.Println(price)
	fmt.Println(score)
}
```

如果没有显式声明小数类型，Go 默认会推断为 **`float64`**。

```go
package main

import "fmt"

func main() {
	pi := 3.14
	fmt.Printf("%v 的类型是 %T\n", pi, pi)
}
```

**注意：**浮点数适合表示近似小数，不适合直接做金额等高精度计算。

## 5. 复数型

Go 也支持复数类型，常见的是 `complex64` 和 `complex128`。

```go
package main

import "fmt"

func main() {
	var c1 complex64 = 1 + 2i
	var c2 complex128 = 3 + 4i

	fmt.Println(c1)
	fmt.Println(c2)
}
```

**了解即可：**初学阶段使用复数的场景不多，先知道 Go 原生支持复数即可。

## 6. 布尔类型

布尔型只有两个值：**`true`** 和 **`false`**。

```go
package main

import "fmt"

func main() {
	var passed bool = true
	var failed bool

	fmt.Println(passed)
	fmt.Println(failed)
}
```

布尔类型需要注意：

- `bool` 的零值是 `false`。
- Go 不允许把整数直接当作布尔值使用。
- `bool` 不能参与数值运算。

```go
// 错误示例：Go 中不能这样写
// if 1 {
// 	fmt.Println("ok")
// }
```

**重点：**条件判断必须得到一个明确的 `bool` 结果。

## 7. 字符相关类型：byte 和 rune

Go 没有单独的 `char` 类型。表示字符时，常用的是：

- `byte`：`uint8` 的别名，常用于处理 ASCII 字符或原始字节。
- `rune`：`int32` 的别名，常用于表示 Unicode 字符。

```go
package main

import "fmt"

func main() {
	var c1 byte = 'a'
	var c2 rune = '中'

	fmt.Println(c1)
	fmt.Println(c2)
	fmt.Printf("%c\n", c1)
	fmt.Printf("%c\n", c2)
}
```

直接打印字符变量时，看到的可能是数字，因为字符本质上对应一个编码值。

```go
package main

import "fmt"

func main() {
	var c = 97
	fmt.Printf("%c\n", c)
}
```

**重点：**想按字符显示时，使用 `%c` 格式化输出。

## 8. 字符串

字符串使用双引号 `"` 定义。

```go
package main

import "fmt"

func main() {
	var s string = "枫枫知道"
	fmt.Println(s)
}
```

字符和字符串的区别：

```go
var c = 'a'  // 字符，使用单引号
var s = "a"  // 字符串，使用双引号
```

**注意：**单引号表示字符，双引号表示字符串，不能混用。

## 9. 多行字符串

反引号可以定义多行字符串，里面的换行和空格会被保留。

```go
package main

import "fmt"

func main() {
	var s = `今天
天气
真好`

	fmt.Println(s)
}
```

**重点：**多行文本、SQL、简单模板内容可以考虑使用反引号字符串。

## 10. 常见转义字符

双引号字符串中可以使用转义字符。

```go
package main

import "fmt"

func main() {
	fmt.Println("枫枫\t知道")
	fmt.Println("枫枫\n知道")
	fmt.Println("\"枫枫\"知道")
	fmt.Println("枫枫\r知道")
	fmt.Println("C:\\pprof\\main.exe")
}
```

常见转义字符：

| 转义字符 | 含义 |
| --- | --- |
| `\t` | 制表符，相当于 Tab |
| `\n` | 换行 |
| `\"` | 输出双引号 |
| `\r` | 回到行首 |
| `\\` | 输出反斜杠 |

**注意：**Windows 路径里的 `\` 要写成 `\\`，否则可能被当作转义字符。

## 11. 基本类型的零值

如果基本数据类型只声明不赋值，变量会得到对应类型的**零值**。

```go
package main

import "fmt"

func main() {
	var a1 int
	var a2 float32
	var a3 string
	var a4 bool

	fmt.Printf("%#v\n", a1)
	fmt.Printf("%#v\n", a2)
	fmt.Printf("%#v\n", a3)
	fmt.Printf("%#v\n", a4)
}
```

常见零值：

| 类型 | 零值 |
| --- | --- |
| `int`、`int8` 等整数 | `0` |
| `float32`、`float64` | `0` |
| `string` | `""` |
| `bool` | `false` |

**重点：**零值不是错误值，而是 Go 自动给变量准备的默认值。

## 12. 本节易错点

- **`uint` 不能保存负数**。
- **`int8`、`uint8` 等小范围类型不能保存超出范围的值**。
- **没有显式声明时，整数通常推断为 `int`，小数通常推断为 `float64`**。
- **Go 没有单独的 `char` 类型**，常用 `byte` 和 `rune` 表示字符相关数据。
- **字符用单引号，字符串用双引号**。
- **反引号字符串会保留原始换行和空格**。
- **布尔值不能和整数互相替代**。
