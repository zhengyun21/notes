# 04 - 判断语句

## 1. if 判断

`if` 用来根据条件执行不同代码。Go 的条件外面**不用写括号**。

```go
package main

import "fmt"

func main() {
	age := 18

	if age >= 18 {
		fmt.Println("成年人")
	}
}
```

## 2. if else

多个分支用 `else if` 和 `else`。

```go
package main

import "fmt"

func main() {
	age := 20

	if age <= 0 {
		fmt.Println("未出生")
	} else if age < 18 {
		fmt.Println("未成年")
	} else if age <= 35 {
		fmt.Println("青年")
	} else {
		fmt.Println("中年")
	}
}
```

**重点：** 条件必须是 `bool`，不能用 `1`、`0` 代替真假。

## 3. if 中声明变量

`if` 可以在条件前先声明一个临时变量。

```go
package main

import "fmt"

func main() {
	if score := 90; score >= 60 {
		fmt.Println("及格")
	} else {
		fmt.Println("不及格")
	}
}
```

`score` := 90; 为自己初始化的变量 ，

`score` >= 60 是 if 的条件。

**注意：**`score` 只在这个 `if else` 结构内部有效。

## 4. switch 枚举判断

`switch` 适合处理多个固定值。

```go
package main

import "fmt"

func main() {
	week := 6

	switch week {
	case 1:
		fmt.Println("周一")
	case 2:
		fmt.Println("周二")
	case 3:
		fmt.Println("周三")
	case 4:
		fmt.Println("周四")
	case 5:
		fmt.Println("周五")
	case 6, 7:
		fmt.Println("周末")
	default:
		fmt.Println("错误")
	}
}
```

**重点：** Go 的 `switch` 默认每个 `case` 后会自动结束，不需要写 `break`。

## 5. switch 条件判断

`switch` 后面也可以不写变量，直接在 `case` 中写条件。

```go
package main

import "fmt"

func main() {
	age := 20

	switch {
	case age <= 0:
		fmt.Println("未出生")
	case age < 18:
		fmt.Println("未成年")
	case age <= 35:
		fmt.Println("青年")
	default:
		fmt.Println("中年")
	}
}
```

这种写法可以替代一长串 `if else`。

## 6. fallthrough

`fallthrough` 会强制继续执行下一个 `case`。

```go
package main

import "fmt"

func main() {
	age := 20

	switch {
	case age <= 0:
		fmt.Println("未出生")
		fallthrough
	case age < 18:
		fmt.Println("未成年")
		fallthrough
	case age <= 35:
		fmt.Println("青年")
	default:
		fmt.Println("中年")
	}
}
```

**注意：**`fallthrough` 不会重新判断下一个 `case` 的条件，初学时少用。

## 7. 易错点

- **`if` 条件不用括号，但 `{}` 不能省略**。
- **条件必须是 `bool`**。
- **`switch case` 默认自动结束，不需要 `break`**。
- **多个值匹配同一个分支，用逗号分隔**：`case 6, 7:`。
- **`fallthrough` 会直接进入下一个分支，不会再次判断条件**。
