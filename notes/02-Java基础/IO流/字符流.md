2. **FileReader**
	1. 创建对象
		1. `public FileReader(File file,)`
		2. `public FileReader(String pathname)`
			1. 字符流输出和字符流输入都有一个 **8192 字节的缓冲区**，当缓冲区满了就会自动将数据写入目的地
	2. 读取数据
		1. `public int read()`
		    1. 读取数据，读到末尾返回-1
		2. `public int read(char[] buffer)`
		    - 读取多个数据，读到末尾返回-1
		    - 读取数据，解码，强制转换三个步骤合并了，把强转之后的字符放到数组当中
		    - 使用`public String(char value[], int offset, int count)`转换为字符串
		3. `public void flush()`
		    - 刷新之后，还可以继续往文件中写出数据

3. **FileWriter**
	1. 创建对象
		1. `public FIleWriter(File file, Boolean append)`
	    2. `public FIleWriter(String pathname, Boolean append)`
		    1. 如果文件不存在，就直接报错
		    2. boolean append 表示对文件进行true追加还是false覆盖
	2. 输出数据
		1. - `void write(int c)`
			- 写出一个字符
		- `void write(String str)`
		    - 写出一个字符串
		- `void write(String str, int off, int len)`
		    - 写出一个字符串的一部分
		- `void write(char[ ] cbuf)`
		    - 写出一个字符数组
		- `void write(char[ ] cbuf, int off, int len)`
		    - 写出字符数组的一部分
