1. 字节缓冲流
	1. 创建一个 size 字节大小的字节缓存输入流（把基本流包装为高级流）
	    - `public BufferedInputStream(InputStream in, int size)`
	2. 创建一个 size 字节大小的字节缓存输出流（把基本流包装为高级流）
	    - `public BufferedOutputStream(OutputStream out, int size)`
	3. 读写入一个字节的数据
	    - `read() // write(int c)`
	4. 读写入多个字节的数据
	    - `read(byte[] bytes) // write(bytes, 0, len)`

2. 字符缓冲流
	1.  创建一个`size*2`字节大小的字符**缓存输入流**(因为 char 类型在 Java 中的大小是两字节) 
		1. public BufferedReader(Reader r)`
		2. 特有方法
			1. `public String readLine()`
				1. 读一整行
				2. 该方法不会将 换行符 读入到缓冲区中
				3. 读到结尾的时候，该方法返回 null
	2.  创建一个`size*2`字节大小的字符**缓存输出流**(因为 char 类型在 Java 中的大小是两字节) 
		1. `public BufferedWriter(Writer r)`
		2. 特有方法
			1. `public void newLine()`
				1. 会根据不同的操作系统写入一个换行符
