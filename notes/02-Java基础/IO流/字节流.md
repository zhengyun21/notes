	2. 字符流：只能传输纯文本文件

	1. 随用随创建（操作不当可能会覆盖文件）
	2. 什么时候不用就关闭
5. **字节流**
	1. **FileInputStream**：字节输入流
		1. 创建对象
			1. `public FileInputStream(File file)`
		    2. `public FileInputStream(String pathname)`
			    1.  如果文件不存在，就直接报错.
		2. 读数据
			1. `public int read( )`
				1. 一次读一个字节数据，对应ASCII上的数字.
				2. 每调用一次移动一次指针，指针到末尾时输出-1.
			2. `public int read(byte[] buffer)`
				1. 一次读取多个字节数据，取决于字节数组大小
				2. 使用`public String(char value[], int offset, int count)`转换为字符串
				3. 读取结束后，会返回-1
		3. 释放资源
		4. 示例![[FileInputStream示例.png]]
	2. **FileOutputStream**：字节输出流
		1. 创建对象
			1.  `public FileOutputStream(File file，Boolean append)`
			2.  `public FileOutputStream(String pathname，Boolean append)`
				1.  文件不存在，直接报错；如果写入参数 true 那么就会在原有文件内继续添加；文件不存在会创建一个新的文件，前提是父级路径是存在的
		2. 写出数据
			1. public void write(int b)
				1. 一次写一个字节数据
				2. 参数是整数，写到本地文件中的是整数在ASCII上对应的字符
			2. void write(byte[] b)
				1. 一次写一个字节数组长度数据
			3. void write(byte[] b, int off, int len)
				1. 一次写一个字节数组的部分数据
		3. 释放资源
