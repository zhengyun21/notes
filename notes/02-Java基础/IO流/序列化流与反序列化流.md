序列化流：
1.创建对象 
	1. Student stu = new Student("zhangsan",23); 
2.创建序列化流的对象/对象操作输出流 
	2. ObjectOutputStream oos = new ObjectOutputStream(new FileOutputStream("student_objet.txt"));			
		1. 对象必须要实现 Serializable 接口 （这个接口只是一个标记性接口，里面没有抽象方法，只表示当前的类可以被序列化）， 如果没有实现接口，就会抛出 NotSerializableException 异常
3.写出数据
	3. oos.writeObject (stu); 
4.释放资源 
	4. oos.close();

反序列化流：
1. 把基本流变成高级流`
	1. public ObjectInputStream(InputStream out)`
2. 把文件反序列化（读入）到程序中去
	1. `public Object readObject()`
	    - 对象必须要实现 Serializable 接口 （这个接口只是一个标记性接口，里面没有抽象方法，只表示当前的类可以被反序列化）， 如果没有实现接口，就会抛出 deserialization 异常
3. 释放资源
	1. `public void close()`
