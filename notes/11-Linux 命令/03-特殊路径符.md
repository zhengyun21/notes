# 特殊路径符

> Linux 中除了具体的绝对路径和相对路径外，还有一些特殊符号用来快速表示特定路径。

---

## 特殊路径符一览

| 符号 | 名称 | 含义 | 示例 |
|------|------|------|------|
| `.` | 当前目录 | 表示当前所在的目录 | `cd ./bin`（进入当前目录下的 bin） |
| `..` | 上级目录 | 表示当前目录的父目录 | `cd ..`（返回上一级） |
| `~` | 家目录 | 当前用户的家目录（Home） | `cd ~` 或 `ls ~/Documents` |
| `/` | 根目录 | 文件系统的最顶层 | `cd /` 或 `ls /etc` |
| `-` | 上次目录 | 上一次所在的工作目录 | `cd -`（快速切换回刚才的目录） |
| `*` | 通配符 | 匹配任意字符（任意长度） | `ls *.txt`（列出所有 txt 文件） |
| `?` | 通配符 | 匹配任意单个字符 | `ls file?.txt`（匹配 file1.txt 等） |
| `~用户名` | 某用户的家目录 | 指定用户的家目录 | `cd ~root` |

---

## 实用示例

```bash
# 查看当前目录下的所有文件（. 表示当前目录）
ls .

# 返回上级目录
cd ..

# 返回上级的上级
cd ../..

# 切换到用户的家目录
cd ~

# 查看家目录下的 Documents 文件夹
ls ~/Documents

# 切换到上次所在的目录
cd -

# 列出当前目录下所有 .md 文件
ls *.md

# 列出当前目录下所有 file 开头、后面跟一个字符的 txt 文件
ls file?.txt
```

---

## 路径拼接技巧

```bash
# 从当前目录进入子目录 bin
cd ./bin
# 等价于
cd bin

# 从当前目录进入同级目录的另一文件夹
cd ../another_folder

# 从 /var/log 进入 /etc/nginx
cd ../../etc/nginx
```

> **记忆口诀**：`.` 是当前，`..` 是上级，`~` 是家，`/` 是根，`cd -` 能往返。
