# Linux 常用快捷键

> 掌握终端快捷键能大幅提升命令行操作效率，减少鼠标依赖。

---

## 光标移动

| 快捷键 | 作用 |
|--------|------|
| `Ctrl + a` | 光标移到**行首**（a = ahead） |
| `Ctrl + e` | 光标移到**行尾**（e = end） |
| `Ctrl + f` | 光标**向前**移动一个字符（f = forward） |
| `Ctrl + b` | 光标**向后**移动一个字符（b = backward） |
| `Alt + f` | 光标向前移动一个**单词** |
| `Alt + b` | 光标向后移动一个**单词** |

---

## 编辑操作

| 快捷键 | 作用 |
|--------|------|
| `Ctrl + d` | 删除光标所在位置的字符（等价于 Delete） |
| `Ctrl + h` | 删除光标前一个字符（等价于 Backspace） |
| `Ctrl + w` | 删除光标前的一个**单词** |
| `Ctrl + k` | 删除从光标到**行尾**的所有内容 |
| `Ctrl + u` | 删除从光标到**行首**的所有内容 |
| `Ctrl + y` | **粘贴**最后一次被删除的内容（yank） |
| `Ctrl + t` | 交换光标前后两个字符的位置 |
| `Alt + t` | 交换光标前后两个单词的位置 |
| `Alt + u` | 将光标后的单词变为大写 |
| `Alt + l` | 将光标后的单词变为小写 |
| `Alt + c` | 将光标后的单词首字母大写 |

---

## 历史命令与搜索

| 快捷键 | 作用 |
|--------|------|
| `Ctrl + r` | **反向搜索**历史命令（按 `Ctrl + r` 继续向上搜索） |
| `Ctrl + g` | 取消搜索，恢复原命令行 |
| `Ctrl + p` | 显示上一条命令（p = previous） |
| `Ctrl + n` | 显示下一条命令（n = next） |
| `!!` | 执行上一条命令 |
| `!n` | 执行历史记录中的第 n 条命令 |
| `!-n` | 执行倒数第 n 条命令 |
| `!关键词` | 执行最近一条以"关键词"开头的命令 |

---

## 终端控制

| 快捷键 | 作用 |
|--------|------|
| `Ctrl + l` | **清屏**（等价于 `clear` 命令，但不会清除历史） |
| `Ctrl + s` | 暂停屏幕输出（锁定滚动） |
| `Ctrl + q` | 恢复屏幕输出（解锁滚动） |
| `Ctrl + c` | **中断**当前运行的命令（发送 SIGINT） |
| `Ctrl + z` | **挂起**当前进程（放入后台，可用 `fg` 恢复） |
| `Ctrl + d` | 退出当前终端（如果命令行为空） |
| `Ctrl + s` / `Ctrl + q` | 暂停/恢复终端输出 |

## 实用技巧

```bash
# 快速复制上一条命令的最后一个参数
Esc + .        # 按 Esc 后按 .（点号）
Alt + .        # 部分终端支持

# 快速输入 !! 执行上一条命令，然后修改
!!:s/old/new/  # 替换上一条命令中的 old 为 new

# 快速使用上一条命令的参数
!$             # 上一条命令的最后一个参数
!*             # 上一条命令的所有参数
!^             # 上一条命令的第一个参数
```

## 记忆口诀

| 场景 | 快捷键 |
|------|--------|
| 想回到开头 | `Ctrl + a` |
| 想去结尾 | `Ctrl + e` |
| 想删整行 | `Ctrl + u`（删到开头）或 `Ctrl + k`（删到结尾） |
| 想清屏幕 | `Ctrl + l` |
| 想停程序 | `Ctrl + c` |
| 想搜历史 | `Ctrl + r` |
| 想暂停程序 | `Ctrl + z` |
