# nmap：网络扫描与安全审计

> `nmap`（Network Mapper）是网络发现和安全审计的强大工具，可以扫描端口、探测服务、识别操作系统等。

---

## 基本语法

```bash
nmap [选项] 目标
```

> **注意**：nmap 需要 root 权限才能执行完整的扫描功能。

---

## 常用扫描类型

| 选项 | 扫描类型 | 说明 |
|------|----------|------|
| `-sS` | TCP SYN 扫描（半开放） | 最常用，速度快，隐蔽性好 |
| `-sT` | TCP 连接扫描 | 完整连接，较慢但更可靠 |
| `-sU` | UDP 扫描 | 扫描 UDP 端口 |
| `-sP` | Ping 扫描 | 只检测主机是否存活 |
| `-sV` | 版本探测 | 检测服务版本 |
| `-O` | 操作系统探测 | 尝试识别目标操作系统 |
| `-A` | 全面扫描 | 启用操作系统、版本探测和脚本扫描 |

---

## 常用选项

| 选项 | 作用 |
|------|------|
| `-p 端口` | 指定扫描端口 |
| `-p-` | 扫描所有 65535 个端口 |
| `-p 1-1000` | 扫描端口范围 |
| `-Pn` | 跳过 Ping 探测（假设主机在线） |
| `-T 0-5` | 设置扫描速度（0=最慢，5=最快） |
| `-v` | 显示详细过程 |
| `-oN 文件` | 将结果保存为普通文本 |
| `-oX 文件` | 将结果保存为 XML 格式 |
| `--top-ports N` | 扫描最常见的 N 个端口 |

---

## 示例

```bash
# 扫描单个主机的常用端口（默认扫描 1000 个常用端口）
nmap 192.168.1.1

# 扫描指定端口
nmap -p 22,80,3306,8080 192.168.1.1

# 扫描端口范围
nmap -p 1-1000 192.168.1.1

# 扫描所有端口
nmap -p- 192.168.1.1

# 扫描并探测服务版本
nmap -sV 192.168.1.1

# 全面扫描（版本 + 操作系统）
nmap -A 192.168.1.1

# 快速扫描（最常用组合）
nmap -sS -Pn -T4 192.168.1.1

# 扫描整个网段
nmap 192.168.1.0/24

# 扫描多个主机
nmap 192.168.1.1 192.168.1.2 192.168.1.3

# 扫描存活主机（Ping 扫描）
nmap -sP 192.168.1.0/24

# 只扫描开放端口，不做其他探测
nmap -sS -Pn -T4 --top-ports 1000 192.168.1.1
```

## 输出解读

```
PORT     STATE SERVICE
22/tcp   open  ssh
80/tcp   open  http
3306/tcp open  mysql
```

| 字段 | 含义 |
|------|------|
| `PORT` | 端口号和协议 |
| `STATE` | 端口状态（open/open\|filtered/closed/filtered） |
| `SERVICE` | 推测的服务名称 |

### 端口状态

| 状态 | 含义 |
|------|------|
| `open` | 端口开放，服务正在监听 |
| `closed` | 端口关闭，无服务监听 |
| `filtered` | 被防火墙过滤，无法确定状态 |
| `unfiltered` | 端口响应了扫描，但无法确定是否开放 |

---

## 安全与法律提醒

> ⚠️ **重要**：nmap 是网络管理员和安全专家的合法工具。但**未经许可扫描他人网络或服务器可能违反法律**。请只在以下场景使用：
> - 扫描自己的服务器
> - 扫描获得授权的网络
> - 学习和安全测试

## 快速对比

| 需求 | 命令 |
|------|------|
| 扫描主机常用端口 | `nmap IP` |
| 扫描指定端口 | `nmap -p 22,80,3306 IP` |
| 扫描所有端口 | `nmap -p- IP` |
| 探测服务版本 | `nmap -sV IP` |
| 扫描网段存活主机 | `nmap -sP 192.168.1.0/24` |
| 快速扫描 | `nmap -sS -Pn -T4 IP` |
| 全面扫描 | `nmap -A IP` |
| 保存扫描结果 | `nmap -oN result.txt IP` |

> **安装**：`sudo apt install nmap` 或 `sudo yum install nmap`。
