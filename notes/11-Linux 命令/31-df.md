# df：磁盘空间查看

> `df`（Disk Free）用于查看文件系统的磁盘空间使用情况，包括总容量、已用、可用和挂载点等信息。

---

## 基本语法

```bash
df [选项] [文件或目录]
```

---

## 常用选项

| 选项 | 全称 | 作用 |
|------|------|------|
| `-h` | human-readable | 以人类可读格式显示（K/M/G） |
| `-H` | si | 以 1000 为基数显示（而非 1024） |
| `-T` | print-type | 显示文件系统类型 |
| `-i` | inodes | 显示 inode 使用情况 |
| `-t 类型` | type | 只显示指定类型的文件系统 |
| `-x 类型` | exclude-type | 排除指定类型的文件系统 |
| `-a` | all | 显示所有文件系统（包括虚拟文件系统） |

---

## 示例

```bash
# 查看所有文件系统的磁盘空间（人类可读）
df -h

# 查看文件系统类型
df -hT

# 查看指定目录所在的文件系统
df -h /var

# 查看 inode 使用情况
df -hi

# 只查看 ext4 类型的文件系统
df -h -t ext4

# 排除 tmpfs
df -h -x tmpfs -x devtmpfs
```

## 输出解读

```
Filesystem     Size  Used Avail Use% Mounted on
/dev/sda1       50G   20G   28G  42% /
```

| 字段 | 含义 |
|------|------|
| `Filesystem` | 文件系统设备 |
| `Size` | 总容量 |
| `Used` | 已用空间 |
| `Avail` | 可用空间 |
| `Use%` | 使用率 |
| `Mounted on` | 挂载点 |

> **注意**：`Avail` 通常不等于 `Size - Used`，因为 ext 系列文件系统会预留一部分空间给 root 用户（默认 5%）。

---

## 与 du 的区别

| 命令 | 作用 | 使用场景 |
|------|------|----------|
| `df` | 查看文件系统级别的磁盘使用 | 看磁盘是否快满了 |
| `du` | 查看目录/文件级别的磁盘使用 | 看哪个目录占空间 |

```bash
# 查看当前目录下各子目录的大小
du -sh *

# 查看当前目录总大小
du -sh .

# 查看最大目录（排序）
du -sh * | sort -hr | head -n 10
```

---

## 快速对比

| 需求 | 命令 |
|------|------|
| 查看磁盘空间 | `df -h` |
| 查看文件系统类型 | `df -hT` |
| 查看 inode 使用 | `df -hi` |
| 查看目录大小 | `du -sh 目录` |
| 查找大目录 | `du -sh * \| sort -hr \| head -n 10` |
| 查看指定目录的磁盘 | `df -h /var` |

> **排查建议**：当磁盘空间不足时，先用 `df -h` 找到哪个文件系统满了，再用 `du -sh * | sort -hr` 找到具体是哪个目录占用了空间。
