# 压缩与解压缩：tar、gzip、zip

> Linux 中常用的压缩格式包括 `.tar`（打包）、`.tar.gz`（打包并压缩）、`.zip` 等。`tar` 是最常用的打包工具。

---

## 1. tar：打包与压缩

`tar`（Tape Archive）可以将多个文件/目录打包成一个文件，并可选择压缩。

### 常用选项

| 选项 | 全称 | 作用 |
|------|------|------|
| `-c` | create | **创建**新的归档文件 |
| `-x` | extract | **解压/提取**归档文件 |
| `-t` | list | **列出**归档文件内容 |
| `-v` | verbose | 显示过程信息 |
| `-f` | file | 指定归档文件名（必须放在最后） |
| `-z` | gzip | 用 gzip 压缩/解压（生成 `.tar.gz` 或 `.tgz`） |
| `-j` | bzip2 | 用 bzip2 压缩/解压（生成 `.tar.bz2`） |
| `-J` | xz | 用 xz 压缩/解压（生成 `.tar.xz`） |
| `-C` | directory | 指定解压目录 |
| `-p` | preserve | 保留文件权限 |

---

### 1.1 压缩（打包）

```bash
# 仅打包（不压缩）
tar -cvf archive.tar 文件1 文件2 目录/

# 打包并用 gzip 压缩（最常用）
tar -czvf archive.tar.gz 文件1 文件2 目录/

# 打包并用 bzip2 压缩（压缩率更高）
tar -cjvf archive.tar.bz2 文件1 文件2 目录/

# 打包并用 xz 压缩（压缩率最高，最慢）
tar -cJvf archive.tar.xz 文件1 文件2 目录/
```

![[Pasted image 20260703145016.png]]

> **记忆口诀**：`czvf` = create（c）+ gzip（z）+ verbose（v）+ file（f）。

---

### 1.2 解压

```bash
# 解压 .tar（仅打包）
tar -xvf archive.tar

# 解压 .tar.gz（最常用）
tar -xzvf archive.tar.gz

# 解压 .tar.bz2
tar -xjvf archive.tar.bz2

# 解压 .tar.xz
tar -xJvf archive.tar.xz

# 解压到指定目录
tar -xzvf archive.tar.gz -C /target/directory/

# 不解压，只查看压缩包内容
tar -tzvf archive.tar.gz
```

![[Pasted image 20260703145050.png]]

> **自动识别格式**：新版本的 tar 可以自动识别压缩格式，直接用 `tar -xvf archive.tar.gz` 也可以。

## 2. gzip / gunzip：单文件压缩

`gzip` 只能压缩单个文件，不能压缩目录。压缩后会删除原文件，生成 `.gz` 文件。

```bash
# 压缩单个文件（原文件会被删除，生成 file.txt.gz）
gzip file.txt

# 解压（file.txt.gz 会被删除，生成 file.txt）
gunzip file.txt.gz
# 或
gzip -d file.txt.gz

# 压缩时保留原文件
gzip -k file.txt

# 查看压缩文件信息
gzip -l file.txt.gz
```

---

## 3. zip / unzip：跨平台压缩

`.zip` 格式在 Windows 和 Linux 之间通用，适合跨平台传输。

```bash
# 压缩文件（-r 递归压缩目录）
zip -r archive.zip 文件1 文件2 目录/

# 压缩时不包含目录本身（只包含目录内容）
zip -r archive.zip 目录/*

# 解压
unzip archive.zip

# 解压到指定目录
unzip archive.zip -d /target/directory/

# 查看压缩包内容（不解压）
unzip -l archive.zip

# 解压时覆盖已有文件（不提示）
unzip -o archive.zip
```

---

## 4. 压缩格式对比

| 格式 | 命令 | 压缩率 | 速度 | 跨平台 | 适用场景 |
|------|------|--------|------|--------|----------|
| `.tar` | `tar -cvf` | 无 | 快 | Linux 为主 | 仅打包 |
| `.tar.gz` | `tar -czvf` | 中等 | 快 | 通用 | **最常用** |
| `.tar.bz2` | `tar -cjvf` | 高 | 慢 | 通用 | 需要更小体积 |
| `.tar.xz` | `tar -cJvf` | 最高 | 最慢 | 通用 | 极致压缩 |
| `.zip` | `zip -r` | 中等 | 快 | **最好** | Windows 交互 |
| `.gz` | `gzip` | 中等 | 快 | 通用 | 单文件压缩 |

---

## 5. 快速参考

| 需求 | 命令 |
|------|------|
| 打包并压缩 | `tar -czvf archive.tar.gz 文件/目录` |
| 解压 tar.gz | `tar -xzvf archive.tar.gz` |
| 查看压缩包内容 | `tar -tzvf archive.tar.gz` |
| 解压到指定目录 | `tar -xzvf archive.tar.gz -C /目标目录` |
| 压缩为 zip | `zip -r archive.zip 文件/目录` |
| 解压 zip | `unzip archive.zip` |
| 单文件压缩 | `gzip file.txt` |
| 单文件解压 | `gunzip file.txt.gz` |

> **注意**：压缩目录时，`tar` 默认会保留目录结构。如果不想保留最外层目录，可以在打包时使用 `tar -czvf archive.tar.gz -C /父目录 目标目录`。
