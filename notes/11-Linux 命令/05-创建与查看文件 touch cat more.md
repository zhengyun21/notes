# touch、cat、more：创建与查看文件

> `touch` 用于创建空文件或更新文件时间戳，`cat` 和 `more` 用于查看文件内容。

---

## 1. touch：创建空文件或更新时间戳

### 基本语法

```bash
touch [选项] 文件名
```

### 示例

```bash
# 创建一个空文件
touch newfile.txt

# 创建多个空文件
touch file1.txt file2.txt file3.txt

# 更新文件的时间戳（不改变内容）
touch existing.txt

# 创建带路径的文件（如果目录不存在会报错，需配合 mkdir -p）
touch /tmp/test/newfile.txt
```

> **注意**：`touch` 不修改文件内容，只创建文件或更新其访问/修改时间。

---

## 2. cat：查看文件内容（适合小文件）

### 基本语法

```bash
cat [选项] 文件名
```

### 常用选项

| 选项 | 作用 |
|------|------|
| `-n` | 显示行号 |
| `-b` | 显示行号（空行不编号） |
| `-s` | 将多个连续空行合并为一行 |
| `-E` | 在行行尾显示 `$` 符号 |

### 示例

```bash
# 查看文件内容
cat file.txt

# 查看文件并显示行号
cat -n file.txt

# 合并多个文件内容输出
cat file1.txt file2.txt

# 合并多个文件到一个新文件
cat file1.txt file2.txt > combined.txt
```

> **注意**：`cat` 会一次性输出全部内容，对于大文件会刷屏，不适合查看长文件。

---

## 3. more：分页查看文件内容（适合大文件）

### 基本语法

```bash
more 文件名
```

### 使用技巧

| 按键 | 作用 |
|------|------|
| `空格` | 向下翻一页 |
| `Enter` | 向下翻一行 |
| `b` | 向上翻一页 |
| `q` | 退出 |
| `/关键词` | 搜索关键词 |
| `n` | 查找下一个匹配 |

### 示例

```bash
# 分页查看日志文件
more /var/log/syslog

# 查看大文件
more largefile.txt
```

> **提示**：`more` 只能向下翻页，无法向上回滚。如果需要双向翻页，推荐使用 `less` 命令。

---

## 4. 快速对比

| 需求 | 推荐命令 | 原因 |
|------|----------|------|
| 创建空文件 | `touch` | 最简单 |
| 查看小文件 | `cat` | 快速输出全部内容 |
| 查看大文件 | `more` 或 `less` | 分页查看，不刷屏 |
| 查看带行号 | `cat -n` | 简洁明了 |
| 可双向翻页 | `less` | 比 `more` 更强大（`less` 是 `more` 的增强版） |
