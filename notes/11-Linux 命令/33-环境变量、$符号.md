# 环境变量与 $ 符号

> 环境变量是 Linux 系统中存储配置信息的键值对，影响进程的运行环境。`$` 符号用于引用变量。

---

## 1. 环境变量

### 查看环境变量

```bash
# 查看所有环境变量
env

# 查看特定环境变量
echo $PATH
echo $HOME
echo $USER

# 使用 printenv
printenv PATH
```

### 常用环境变量

| 变量名 | 含义 | 示例值 |
|--------|------|--------|
| `PATH` | 命令搜索路径 | `/usr/local/bin:/usr/bin:/bin` |
| `HOME` | 当前用户的家目录 | `/home/novice21` |
| `USER` | 当前用户名 | `novice21` |
| `SHELL` | 当前登录 Shell | `/bin/bash` |
| `PWD` | 当前工作目录 | `/home/novice21/projects` |
| `LANG` | 系统语言/编码 | `zh_CN.UTF-8` |
| `HOSTNAME` | 主机名 | `myserver` |
| `PS1` | 命令行提示符格式 | `\u@\h:\w\$` |

## 2. 设置环境变量

### 临时设置（仅当前终端有效）

```bash
# 设置环境变量
export MY_VAR="hello"

# 追加到 PATH
export PATH=$PATH:/usr/local/mybin

# 验证
echo $MY_VAR
```

### 永久设置（写入配置文件）

```bash
# 编辑用户级配置文件
vim ~/.bashrc

# 添加
export MY_VAR="hello world"
export PATH=$PATH:/usr/local/mybin

# 使配置生效
source ~/.bashrc
```

### 系统级配置（所有用户生效）

```bash
# 编辑系统级配置文件（需 root 权限）
sudo vim /etc/profile
# 或
sudo vim /etc/environment

# 生效
source /etc/profile
```

> **加载顺序**：`/etc/profile` → `~/.bash_profile` → `~/.bashrc`。不同发行版略有差异。

## 3. $ 符号的用法

`$` 在 Shell 中用于引用变量、执行命令替换等。

| 用法 | 含义 | 示例 |
|------|------|------|
| `$变量名` | 引用变量 | `echo $PATH` |
| `${变量名}` | 引用变量（安全写法，支持拼接） | `echo ${HOME}/file` |
| `$?` | **上一条命令的退出状态** | `echo $?`（0 表示成功） |
| `$0` | 当前脚本名称 | — |
| `$1` ~ `$9` | 脚本的第 1~9 个参数 | `echo $1` |
| `$#` | 脚本参数的个数 | — |
| `$@` | 所有参数（数组形式） | — |
| `$*` | 所有参数（字符串形式） | — |
| `$$` | 当前 Shell 的进程 ID | — |
| `$!` | 最后放入后台的进程 ID | — |

## 4. 变量引用技巧

```bash
# 基本引用
name="Linux"
echo $name

# 安全引用（防止变量名边界问题）
echo ${name}OS        # 输出 LinuxOS
echo $name OS         # 输出 Linux OS（有空格）

# 默认值（变量未设置时的默认值）
echo ${UNSET_VAR:-default}    # 输出 default

# 修改变量（字符串替换）
path="/usr/local/bin"
echo ${path/local/LOCAL}      # 输出 /usr/LOCAL/bin

# 获取字符串长度
str="hello"
echo ${#str}          # 输出 5
```

## 5. 快速对比

| 需求 | 命令/写法 |
|------|-----------|
| 查看所有环境变量 | `env` 或 `printenv` |
| 查看 PATH | `echo $PATH` |
| 临时设置变量 | `export VAR=value` |
| 永久设置变量 | 编辑 `~/.bashrc` 后 `source ~/.bashrc` |
| 检查上条命令是否成功 | `echo $?`（0=成功） |
| 获取脚本参数 | `$1`, `$2`, `$@` |
| 获取参数个数 | `$#` |
| 变量默认值 | `${VAR:-default}` |
| 变量字符串长度 | `${#VAR}` |

> **脚本技巧**：在脚本开头使用 `set -e`，可以在任何命令返回非 0 时立即退出，防止错误继续执行。
