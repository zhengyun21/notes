# ls 与 tree：列出目录内容

> `ls` 是 Linux 中使用频率最高的命令之一，用于查看目录下的文件和文件夹。`tree` 则以树状结构展示目录层级。

---

## 1. ls 命令

### 基本语法

```bash
ls [选项] [路径]
```

> 不指定路径时，默认列出当前目录的内容。

### 常用选项

| 选项 | 全称 | 作用 |
|------|------|------|
| `-a` | all | **显示所有文件**，包括隐藏文件（以 `.` 开头的文件） |
| `-l` | long | **以列表形式展示**，显示详细信息（权限、大小、日期等） |
| `-h` | human-readable | **人类可读格式**，需与 `-l` 组合使用，文件大小显示为 K/M/G |
| `-t` | time | 按修改时间排序，最新的在最前面 |
| `-r` | reverse | 反向排序 |
| `-S` | size | 按文件大小排序，最大的在最前面 |
| `-R` | recursive | 递归列出子目录内容 |
| `-d` | directory | 只列出目录本身，而非目录内的内容 |

### 组合使用示例

```bash
# 显示所有文件（含隐藏文件）
ls -a

# 以详细列表显示，并附带人类可读的文件大小
ls -l -h

# 等价于上面，选项可以合并
ls -lh

# 显示所有文件 + 详细信息 + 人类可读大小
ls -alh

# 按时间倒序排列（最新修改的文件在最上面）
ls -lt

# 递归列出所有文件和子目录
ls -R
```

> **提示**：`-l` 输出中的第一列是权限（如 `drwxr-xr-x`），第二列是硬链接数，第三列是所有者，第四列是所属组，第五列是文件大小，第六列是最后修改时间，最后一列是文件名。

## 2. tree 命令

`tree` 以树状结构展示目录层级，比 `ls -R` 更直观。

### 基本用法

```bash
# 显示当前目录的树状结构
tree

# 只显示前两层
tree -L 2

# 显示所有文件（包括隐藏文件）
tree -a

# 只显示目录
tree -d

# 显示文件大小（人类可读）
tree -h
```

![[Pasted image 20260704140850.png]]

> **注意**：部分最小化安装的 Linux 系统可能未预装 `tree`，可以通过 `yum install tree` 或 `apt install tree` 安装。

---

## 3. 快速对比

| 需求 | 推荐命令 |
|------|----------|
| 快速查看当前目录文件 | `ls` |
| 查看文件详细信息 | `ls -l` 或 `ls -lh` |
| 查看隐藏文件 | `ls -a` |
| 查看目录整体结构 | `tree` 或 `tree -L 2` |
| 查看最近修改的文件 | `ls -lt` |
