# 端口：概念与常用操作

> 端口（Port）是计算机网络中逻辑通信的端点，用于区分同一 IP 地址上的不同服务。

---

## 1. 端口基本概念

### 端口范围

| 范围 | 名称 | 说明 |
|------|------|------|
| 0 - 1023 | **知名端口**（Well-known Ports） | 由 IANA 分配，通常需要 root 权限才能绑定 |
| 1024 - 49151 | **注册端口**（Registered Ports） | 应用程序注册使用 |
| 49152 - 65535 | **动态/私有端口**（Dynamic/Private） | 临时分配给客户端连接 |

### 常用知名端口

| 端口 | 服务 | 说明 |
|------|------|------|
| 20/21 | FTP | 文件传输协议 |
| 22 | SSH | 安全远程登录 |
| 23 | Telnet | 远程登录（不安全） |
| 25 | SMTP | 邮件发送 |
| 53 | DNS | 域名解析 |
| 80 | HTTP | 网页浏览 |
| 110 | POP3 | 邮件接收 |
| 143 | IMAP | 邮件访问 |
| 443 | HTTPS | 加密网页浏览 |
| 3306 | MySQL | 数据库服务 |
| 3389 | RDP | Windows 远程桌面 |
| 5432 | PostgreSQL | 数据库服务 |
| 6379 | Redis | 缓存数据库 |
| 8080 | HTTP Alternate | 备用 HTTP 端口（如 Tomcat） |
| 9200 | Elasticsearch | 搜索引擎 |
| 27017 | MongoDB | 数据库服务 |

## 2. 查看端口占用

### 使用 netstat

```bash
# 查看所有监听的端口
netstat -tlnp

# 查看指定端口
netstat -anp | grep :3306

# 查看 TCP 端口
netstat -tlnp

# 查看 UDP 端口
netstat -ulnp
```

### 使用 ss（推荐，替代 netstat）

```bash
# 查看所有监听的端口
ss -tlnp

# 查看指定端口
ss -tlnp | grep :3306

# 查看所有连接状态
ss -ant

# 查看 UDP 端口
ss -ulnp
```

### 使用 lsof

```bash
# 查看指定端口被哪个进程占用
sudo lsof -i :3306

# 查看所有网络连接
sudo lsof -i
```

## 3. 测试端口连通性

```bash
# 测试 TCP 端口是否开放（使用 telnet）
telnet 192.168.1.1 80

# 测试 TCP 端口（使用 nc/netcat）
nc -vz 192.168.1.1 80

# 测试 UDP 端口
nc -vzu 192.168.1.1 53

# 扫描多个端口
nc -vz 192.168.1.1 80 443 3306
```

## 4. 防火墙端口管理

### firewalld（CentOS/RHEL）

```bash
# 查看防火墙状态
sudo firewall-cmd --state

# 查看开放的端口
sudo firewall-cmd --list-ports

# 开放端口（永久生效）
sudo firewall-cmd --add-port=8080/tcp --permanent

# 关闭端口
sudo firewall-cmd --remove-port=8080/tcp --permanent

# 重载防火墙配置
sudo firewall-cmd --reload
```

### ufw（Ubuntu/Debian）

```bash
# 查看状态
sudo ufw status

# 开放端口
sudo ufw allow 8080/tcp

# 关闭端口
sudo ufw delete allow 8080/tcp

# 开放特定 IP 访问
sudo ufw allow from 192.168.1.100 to any port 3306
```

## 5. 快速对比

| 需求 | 命令 |
|------|------|
| 查看监听端口 | `ss -tlnp` 或 `netstat -tlnp` |
| 查看端口占用进程 | `sudo lsof -i :端口号` |
| 测试端口连通 | `nc -vz IP 端口` |
| 开放防火墙端口 | `sudo firewall-cmd --add-port=8080/tcp --permanent` |
| 查看所有端口 | `cat /etc/services` |

> **注意**：生产环境中开放端口应遵循最小权限原则，只开放必要的端口，并限制可访问的 IP 范围。
