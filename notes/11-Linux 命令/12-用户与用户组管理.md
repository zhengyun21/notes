# 用户与用户组管理

> Linux 是多用户系统，理解用户和用户组的管理是权限控制的基础。

---

## 1. 用户组管理

### 常用命令

| 命令 | 作用 | 示例 |
|------|------|------|
| `groupadd 组名` | 创建用户组 | `sudo groupadd developers` |
| `groupdel 组名` | 删除用户组 | `sudo groupdel developers` |
| `groupmod -n 新名 旧名` | 重命名用户组 | `sudo groupmod -n dev developers` |
| `cat /etc/group` | 查看所有用户组 | `grep developers /etc/group` |
| `getent group 组名` | 查看指定组信息 | `getent group sudo` |

### 示例

```bash
# 创建一个新用户组
sudo groupadd developers

# 查看 /etc/group 文件结构
cat /etc/group
# 输出格式：组名:密码占位符:GID:成员列表
```

## 2. 用户管理

### 常用命令

| 命令 | 作用 | 示例 |
|------|------|------|
| `useradd 用户名` | 创建用户 | `sudo useradd tomcat` |
| `useradd -m 用户名` | 创建用户并创建家目录 | `sudo useradd -m tomcat` |
| `useradd -g 组名 用户名` | 创建用户并指定主组 | `sudo useradd -g developers tom` |
| `useradd -G 组1,组2 用户名` | 创建用户并指定附加组 | `sudo useradd -G sudo,developers tom` |
| `userdel 用户名` | 删除用户（保留家目录） | `sudo userdel tomcat` |
| `userdel -r 用户名` | 删除用户并删除家目录 | `sudo userdel -r tomcat` |
| `passwd 用户名` | 设置/修改密码 | `sudo passwd tomcat` |
| `usermod -aG 组名 用户名` | 将用户加入附加组 | `sudo usermod -aG sudo tomcat` |
| `usermod -g 组名 用户名` | 修改用户主组 | `sudo usermod -g developers tom` |

### 示例

```bash
# 创建用户并设置密码
sudo useradd -m tomcat
sudo passwd tomcat

# 创建用户时指定附加组
sudo useradd -m -G sudo,developers tom

# 将已有用户加入 sudo 组
sudo usermod -aG sudo tom

# 查看用户信息
id tom
```

![[Pasted image 20260630133637.png]]

> **注意**：`usermod -aG` 中的 `-a`（append）不能省略，否则会把用户从其他组中移除！

## 3. getent：查询系统数据库

`getent` 用于查询系统的各种数据库（用户、组、主机、服务等），比直接查看 `/etc/passwd` 或 `/etc/group` 更规范。

### 基本语法

```bash
getent 数据库 [键值]
```

### 示例

```bash
# 查看所有用户
getent passwd

# 查看指定用户信息
getent passwd root

# 查看所有用户组
getent group

# 查看指定组信息
getent group sudo

# 查看 /etc/hosts 中的主机
getent hosts localhost
```

### /etc/passwd 格式说明

```
用户名:密码占位符:UID:GID:注释:家目录:默认Shell
```

例如：`root:x:0:0:root:/root:/bin/bash`

| 字段 | 含义 |
|------|------|
| `root` | 用户名 |
| `x` | 密码占位符（实际密码在 /etc/shadow） |
| `0` | 用户 ID（UID），root 是 0 |
| `0` | 主组 ID（GID） |
| `root` | 注释/全名 |
| `/root` | 家目录 |
| `/bin/bash` | 默认登录 Shell |

## 4. 快速对比

| 需求 | 命令 |
|------|------|
| 创建用户 | `sudo useradd -m 用户名` |
| 删除用户 | `sudo userdel -r 用户名` |
| 修改密码 | `sudo passwd 用户名` |
| 创建用户组 | `sudo groupadd 组名` |
| 用户加入组 | `sudo usermod -aG 组名 用户名` |
| 查看用户信息 | `id 用户名` 或 `getent passwd 用户名` |
| 查看用户所属组 | `groups 用户名` |
| 查看所有用户 | `cat /etc/passwd` 或 `getent passwd` |
