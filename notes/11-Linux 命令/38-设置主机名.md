# 设置主机名

> 主机名（Hostname）是标识 Linux 服务器的名称，用于网络识别和日志记录。

---

## 查看当前主机名

```bash
# 查看主机名
hostname

# 查看详细信息
hostnamectl

# 查看 /etc/hostname 文件
cat /etc/hostname
```

---

## 临时修改（重启失效）

```bash
sudo hostname 新主机名
```

> 仅对当前会话有效，重启后恢复为 `/etc/hostname` 中的配置。

---

## 永久修改（推荐）

### 方法 1：hostnamectl（推荐）

```bash
sudo hostnamectl set-hostname 新主机名
```

### 方法 2：修改配置文件

```bash
# 修改 /etc/hostname
sudo echo "新主机名" > /etc/hostname

# 或
sudo vim /etc/hostname
```

### 修改 /etc/hosts（确保本地解析正常）

```bash
sudo vim /etc/hosts
```

添加或修改：

```
127.0.0.1   新主机名
127.0.1.1   新主机名
```

> `/etc/hosts` 中的配置确保 `ping 新主机名` 和 `sudo` 等命令不会因主机名解析失败而报错。

---

## 验证修改

```bash
# 重启后生效（或注销重新登录）
hostname
hostnamectl
```

---

## 快速对比

| 需求 | 命令 |
|------|------|
| 查看主机名 | `hostname` |
| 临时修改 | `sudo hostname 新名字` |
| 永久修改（推荐） | `sudo hostnamectl set-hostname 新名字` |
| 修改配置文件 | 编辑 `/etc/hostname` 和 `/etc/hosts` |

> **命名建议**：主机名应简洁明了，建议采用 `用途-环境-编号` 格式，如 `web-prod-01`、`db-test-02`。
