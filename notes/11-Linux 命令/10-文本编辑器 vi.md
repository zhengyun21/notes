# vi/vim：文本编辑器

> `vi` 是 Linux 系统自带的经典文本编辑器，`vim` 是其增强版（Vi IMproved）。掌握 vi/vim 是服务器端编辑文件的必备技能。

---

## 1. 三种工作模式

vi/vim 有三种工作模式，编辑时需要在它们之间切换：

| 模式 | 名称 | 作用 | 切换方式 |
|------|------|------|----------|
| **命令模式** | Normal | 默认模式，用于移动光标、复制、删除、查找等 | 按 `Esc` 进入 |
| **输入模式** | Insert | 用于输入和编辑文本 | 按 `i`/`a`/`o` 等进入 |
| **底线命令模式** | Command-line | 用于保存、退出、查找替换、设置等 | 按 `:` 进入 |

### 模式切换图

```
        命令模式（Normal）
        /        \
       /          \
     i,a,o  <---  Esc
     /            \
    /              \
输入模式（Insert）  底线命令模式（:）
```

> **打开文件后的默认状态是命令模式**，不能直接输入文字，需要按 `i` 切换到输入模式。

## 2. 命令模式常用快捷键

命令模式是 vi/vim 的核心，掌握这些快捷键能大幅提升编辑效率。

### 2.1 光标移动

| 按键 | 作用 |
|------|------|
| `h` `j` `k` `l` | 左、下、上、右移动（推荐使用，不用离开主行） |
| `0` | 跳到行首 |
| `$` | 跳到行尾 |
| `gg` | 跳到文件开头 |
| `G` | 跳到文件末尾 |
| `nG` | 跳到第 n 行 |
| `Ctrl + f` | 向下翻一页 |
| `Ctrl + b` | 向上翻一页 |
| `w` | 跳到下一个单词开头 |
| `b` | 跳到上一个单词开头 |

### 2.2 复制、粘贴、删除

| 按键 | 作用 |
|------|------|
| `yy` | 复制当前行 |
| `nyy` | 复制从当前行开始的 n 行 |
| `p` | 在光标后粘贴 |
| `P` | 在光标前粘贴 |
| `dd` | 删除当前行（剪切） |
| `ndd` | 删除从当前行开始的 n 行 |
| `x` | 删除光标所在字符 |
| `dw` | 删除当前单词 |
| `u` | 撤销上一步操作 |
| `Ctrl + r` | 恢复撤销 |

### 2.3 查找

| 按键 | 作用 |
|------|------|
| `/关键词` | 从光标位置向下查找 |
| `?关键词` | 从光标位置向上查找 |
| `n` | 查找下一个匹配 |
| `N` | 查找上一个匹配 |

## 3. 输入模式进入方式

| 按键 | 进入位置 |
|------|----------|
| `i` | 在光标**前**插入（insert） |
| `a` | 在光标**后**插入（append） |
| `o` | 在光标下方**新开一行** |
| `O` | 在光标上方**新开一行** |
| `I` | 在行首插入 |
| `A` | 在行尾插入 |

> 输入完成后按 `Esc` 回到命令模式。

---

## 4. 底线命令模式

在命令模式下按 `:` 进入底线命令模式，光标会跳到屏幕底部。

### 4.1 保存与退出

| 命令 | 作用 |
|------|------|
| `:w` | 保存文件（write） |
| `:w!` | 强制保存 |
| `:q` | 退出（quit） |
| `:q!` | **强制退出，不保存** |
| `:wq` | 保存并退出 |
| `:x` | 保存并退出（同 `:wq`） |
| `:w 文件名` | 另存为 |
| `ZZ` | 快捷键：保存并退出（命令模式下） |
| `ZQ` | 快捷键：不保存退出（命令模式下） |

![[Pasted image 20260629202814.png]]

### 4.2 查找与替换

| 命令 | 作用 |
|------|------|
| `:%s/old/new/` | 替换当前行第一个匹配 |
| `:%s/old/new/g` | 替换当前行所有匹配 |
| `:%s/old/new/gc` | 全局替换，每次确认 |
| `:n1,n2s/old/new/g` | 在第 n1 到 n2 行之间替换 |
| `:%s/old/new/g` | 全文替换所有匹配 |

### 4.3 显示设置

| 命令 | 作用 |
|------|------|
| `:set nu` | 显示行号 |
| `:set nonu` | 取消行号 |
| `:set syntax=on` | 开启语法高亮 |
| `:set nohlsearch` | 取消搜索高亮 |
| `:set paste` | 进入粘贴模式（防止自动缩进混乱） |

---

## 5. 快速入门流程

```
1. vim 文件名        → 打开文件（进入命令模式）
2. 按 i              → 进入输入模式
3. 编辑内容          → 正常输入文字
4. 按 Esc            → 回到命令模式
5. 按 :wq            → 保存并退出
```

---

## 6. 常用对比

| 需求 | 操作 |
|------|------|
| 打开文件 | `vim filename` |
| 开始编辑 | 按 `i` 进入输入模式 |
| 保存 | `:w` |
| 退出 | `:q` |
| 强制退出不保存 | `:q!` |
| 保存并退出 | `:wq` 或 `ZZ` |
| 复制一行 | `yy` |
| 删除一行 | `dd` |
| 粘贴 | `p` |
| 撤销 | `u` |
| 显示行号 | `:set nu` |
| 全文替换 | `:%s/old/new/g` |

> **建议**：在命令模式下尽量使用 `hjkl` 移动光标，这样手不用离开键盘主行，效率更高。初学者可以先用箭头键过渡，逐步习惯 `hjkl`。
