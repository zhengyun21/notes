# cd 与 pwd：目录切换与定位

> `cd`（Change Directory）用于切换当前工作目录，`pwd`（Print Working Directory）用于显示当前所在位置。

---

## 1. cd 命令：切换目录

### 基本语法

```bash
cd [目标路径]
```

### 常用切换方式

| 命令 | 作用 |
|------|------|
| `cd` 或 `cd ~` | 切换到当前用户的**家目录**（Home） |
| `cd /` | 切换到**根目录** |
| `cd ..` | 切换到**上级目录** |
| `cd -` | 切换到**上一次所在的目录**（快速来回切换） |
| `cd /var/log` | 切换到绝对路径 `/var/log` |
| `cd ./local` | 切换到当前目录下的 `local` 子目录 |
| `cd ~用户名` | 切换到指定用户的家目录 |

### 示例

```bash
# 切换到根目录
cd /

# 切换到上级目录
cd ..

# 切换到 /etc 目录
cd /etc

# 快速回到刚才的目录
cd -

# 切换到当前用户的家目录
cd ~
```

> **注意**：`cd` 只改变当前 shell 的工作目录，不会影响其他终端窗口。

---

## 2. pwd 命令：显示当前路径

```bash
pwd
```

输出当前工作目录的**绝对路径**（从根目录 `/` 开始的完整路径）。

### 示例

```bash
$ pwd
/home/novice21/projects
```

---

## 3. 快速对比

| 需求 | 命令 |
|------|------|
| 查看当前在哪 | `pwd` |
| 回到家目录 | `cd` 或 `cd ~` |
| 去根目录 | `cd /` |
| 返回上一级 | `cd ..` |
| 快速往返 | `cd -` |
